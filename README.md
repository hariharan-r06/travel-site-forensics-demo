# Tourist Explorer

A modern, responsive React application for discovering and exploring tourist destinations around the world.

## Features

- **Modern UI Design**: Clean and modern interface with custom CSS styling
- **Responsive Layout**: Fully responsive design that works on all devices
- **Page Navigation**: React Router implementation for seamless navigation
- **Destination Gallery**: Grid layout showcasing various tourist destinations
- **Contact Form**: Interactive contact form with validation
- **About Section**: Comprehensive information about the app and team

## Pages

- **Home**: Welcome section with featured destinations and app features
- **Destinations**: Grid layout of tourist spots with images and descriptions
- **About**: Information about the app, mission, and team
- **Contact**: Contact form and company information

## Technologies Used

- React 18
- Vite
- React Router DOM
- Custom CSS (no external UI frameworks)
- JSX

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser and navigate to `http://localhost:5173`
